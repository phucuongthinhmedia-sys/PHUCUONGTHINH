# @repo/shared-components

Shared React components for the Digital Showroom monorepo.

## Purpose

This package contains reusable React components used across frontend and CMS applications, ensuring consistent UI and reducing code duplication.

## Contents

- UI primitives (Button, Input, Select, etc.)
- Form components with validation
- Modal and Dialog components
- Card, Badge, Alert components

## Usage

```typescript
import { Button, Input, Modal } from "@repo/shared-components";
```

## Development

```bash
# Build components
pnpm build

# Watch mode
pnpm dev
```
